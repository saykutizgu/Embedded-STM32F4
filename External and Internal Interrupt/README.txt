This is an example of External and Internal interrupts with STM32F401RE Board Using Atollic TrueSTUDIO.

For external interrupt, an external LED to digital pin (GPIO A, Pin 8) configuration and USER Button on the board is used.

For internal interrupt, timer 3 (General Purpose Timer) of the board and USER Led (GPIO A, Pin 5) on the board is used.

You can find the code in src/main.c
